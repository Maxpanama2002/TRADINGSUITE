1. Первый запуск

После установки и запуска ты попадёшь на главный экран — Обзор.

Что увидишь:

  • Сайдбар слева — разделы: Обзор, Портфель, Торговля, Анализ, Планирование, Дополнительно
  • Главный экран (Обзор) — настраиваемая сетка из 16 виджетов
  • Профиль внизу слева — клик ведёт в Личный кабинет
  • Topbar — заголовок страницы + кнопка «+ Сделка»
Совет: первым делом зайди в Личный кабинет → Профиль и установи свой торговый стиль (scalper / daytrader / swing / positional / investor). Это влияет на советы AI.

2. Записать первую сделку

Клик «+ Сделка» в topbar или «+ Новая сделка» в Журнале. Откроется страница записи.

Заполни:

  1. Дата и время — авто-сейчас, можно менять
  2. Инструмент — например BTC/USDT
  3. Направление — Long или Short
  4. Плечо (default 1), Margin (Isolated/Cross), MMR — для расчёта ликвидации
  5. Размер в qty или USDT (синхронизированы) · Цена входа · Стоп-лосс
  6. Тейк-профиты — кликни «+ TP» для добавления, можно до 5 штук с %
  7. Эмоция перед входом и Соблюдение плана

Под каждым SL и TP отображается inline P&L — увидишь сразу убыток/прибыль и ROI от маржи. Внизу — сводка позиции с ценой ликвидации.

3. Paper Trading

Сайдбар → Торговля → Paper Trading. У тебя $10,000 виртуальных.

Клик «+ Открыть позицию». Модалка работает как Futures-калькулятор:

  • Long / Short toggle (зелёный/красный)
  • Isolated / Cross margin
  • Символ + Плечо + Цена входа + Размер (qty или USDT)
  • Stop-Loss с inline P&L
  • Multiple Take-Profits с распределением % (auto-balance)

При пересечении SL / TP / Liquidation позиция закрывается автоматически с правильной exit-ценой.

4. AI-ассистент

Сайдбар → AI-ассистент. Перед первым запросом — выбери контекст:

  1. Клик «Контекст» в шапке чата
  2. Отметь разделы которые AI увидит: журнал, дневник, портфель, watchlist, психо, funding+OI, стратегия, цели, идеи, статистика и т.д.
  3. Закрой панель
  4. Спрашивай

Пресеты «Наставник / Коуч / Аналитик / Психолог / Свой» меняют тон ответов.

Master-выключатель в Личном кабинете полностью отрубает контекст — ни один из твоих разделов не пойдёт в AI. Используй когда хочешь спросить что-то «вообще».

5. Funding + OI

Сайдбар → Анализ → Funding+OI. Топ-30 USDT-perp пар с агрегацией по 3 биржам.

Что в таблице:

  • Цена + 24ч изменение
  • OI total (Binance + Bybit + OKX)
  • Funding с цветовой градацией CoinGlass-style
  • L/S ratio — % лонг/шорт аккаунтов с двухцветной полоской
  • Сигнал — алгоритм классифицирует ситуацию (overheat, squeeze, anomaly...)

Клик по строке — drill-down с разбивкой OI/funding/volume по каждой бирже отдельно.

6. Алерты Watchlist

Сайдбар → Анализ → Watchlist. Добавь монеты через input «+ символ».

При пересечении алерта (не «выше» а именно cross):

  • Native macOS push (если permission)
  • In-app toast — большая плашка по центру экрана
  • Звук — двойной beep
  • Dock bounce — иконка прыгает

Алерты работают в фоне даже если ты на другой странице.

7. iCloud sync

Личный кабинет → Быстрые настройки → ☁️ Бэкап и iCloud.

  • ☁️↑ Сохранить в iCloud — пушит все данные в iCloud Drive
  • ☁️↓ Загрузить из iCloud — забирает с iCloud, заменяет локальные
  • Авто-синхронизация — сохраняет каждые 60 секунд

Шифрование AES-256-GCM с PBKDF2 (200K итераций). Ключ ты задаёшь сам — пароль хранится в keychain. Без пароля файл расшифровать нельзя.

8. Supabase (cross-device sync)

Альтернатива iCloud — работает между Mac, Windows, web-версией. Краткая версия:

  1. Создай проект на supabase.com (free tier)
  2. SQL Editor → запусти содержимое db/supabase-schema.sql
  3. Project Settings → API → скопируй URL и anon key
  4. В Trading Suite: Личный кабинет → Быстрые настройки → 🌥 Облако
  5. Вставь URL + anon key → Подключить
  6. На главном устройстве: ☁️↑ Загрузить всё в облако

9. Бэкап и восстановление

Кроме iCloud и Supabase, есть локальный экспорт:

  • ↓ Резервная копия — сохраняет зашифрованный JSON-файл
  • ↑ Восстановить — выбираешь файл, восстанавливает данные

Файл бэкапа шифруется AES-256-GCM. Не теряй пароль — без него файл не открывается.

10. Cooling-off и pre-check

Включаются в Личном кабинете → Правила самоконтроля.

Cooling-off / Tilt-detection:

  • 3 убытка подряд + менее 24ч с последнего → попытка открыть новую сделку → модалка-предупреждение
  • Юзер может «понимаю, всё равно» (override на сутки) или взять паузу

Pre-trade checklist:

  • 5 обязательных пунктов: Stop-Loss установлен, размер ≤2%, согласовано с планом, не на эмоциях, R:R ≥2:1
  • Кнопка «Сохранить» disabled пока не все отмечены

Tilt Detector (AI):

  • В Личном кабинете — кнопка «Проверить» в секции «🧠 AI-проверка состояния»
  • AI читает последние 5 записей дневника
