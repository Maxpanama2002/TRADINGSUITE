Возможности

Полный разбор того, что есть в Trading Suite. От базового журнала до AI tilt-детектора.

📓

Торговый журнал

Полноценная система ведения сделок с психологией и аналитикой.

  • Запись сделки — символ, направление, размер (qty или USDT), плечо, маржа (Isolated/Cross), entry, SL, multiple TPs с %, MMR, депозит, риск%
  • Inline P&L — под каждым TP и SL: убыток/профит и ROI от маржи
  • Цена ликвидации — формула с учётом MMR и режима маржи
  • Эмоции — 10 состояний от спокойствия до тильта
  • Соблюдение плана — Полностью / Частично / Нарушил
  • Pre-trade checklist — 5 обязательных пунктов перед сохранением
  • AI Trade Review — Claude анализирует сделку: оценка A-F, сильные/слабые стороны, главный урок
  • Аналитика — equity curve, винрейт, profit factor, корреляции «эмоция → результат»
  • Spot / Futures — переключатель рынка при записи сделки; фильтр по рынку в Сделках и Аналитике
  • Архив + Периоды — закрытие периодов (месяц/квартал/год) со снимком статистики
🎮

Paper Trading

Виртуальная торговля на $10,000 с live-ценами Binance.

  • Spot / Futures — переключатель рынка; в Spot-режиме скрыты фьючерсные поля (плечо, MMR, ликвидация)
  • Размер в qty или USDT — два связанных поля, синхронизация при изменении цены
  • Multiple TPs — до 5 тейк-профитов с частичным закрытием по %
  • Edit SL/TP — переставлять стопы и тейки на лету; сработавшие TP помечены ✓
  • Авто-закрытие — при достижении SL/TP/Liquidation позиция закрывается с правильной exit-ценой
  • История — каждая сделка с причиной закрытия (SL / TP / Liq / Manual), удаление записей с подтверждением
  • Reset — сброс к стартовым $10K
🧠

AI-ассистент

Claude Sonnet / Opus / Haiku с доступом к твоим реальным данным.

  • 5 пресетов — Наставник, Коуч, Аналитик, Психолог, Свой
  • Выбор контекста — журнал, дневник, портфель, watchlist, психо-журнал, funding+OI, стратегия, цели, идеи, статистика
  • Шаблоны вопросов — 6 готовых промптов для разбора
  • AI Trade Review — отдельная кнопка для разбора конкретной сделки
  • Tilt Detector — AI читает последние записи дневника и определяет состояние: calm / elevated / tilt
  • Защита от prompt injection — данные в тегах <user_data>
📈

Funding + OI Dashboard

Агрегация derivatives-данных с трёх бирж в реальном времени.

  • 3 биржи — Binance, Bybit, OKX (weighted average по OI)
  • Топ-30 пар по объёму торгов USDT-perp
  • BTC и ETH spotlight cards с funding, OI, L/S ratio
  • Drill-down — клик на строку показывает разбивку по биржам
  • Анализ сигналов — 8 сценариев: anomaly, extreme overheat, squeeze setup и т.д.
  • 4 тайминга — 8ч / 24ч / 7д / 30д (множитель ставки)
  • L/S ratio — % лонгов и шортов аккаунтов с двухцветной полоской
  • Auto-refresh каждые 30 секунд пока ты на странице
👁

Watchlist с алертами

Список монет с live-ценами и алертами на пересечение цены.

  • Live-цены с CoinGecko / Binance, 24ч change
  • Алерты на цену — выставил порог, после пересечения срабатывает
  • 4 канала уведомлений одновременно: native push, in-app toast, звук, dock bounce
  • Background watcher — алерты срабатывают даже когда ты на другой странице
  • Cross-detection — fires только при ПЕРЕХОДЕ через цену, без ложных срабатываний
🛡

Защита и психология

Механизмы которые мешают сливать счёт на эмоциях.

  • Cooling-off mode — 3 убытка подряд + менее 24ч с последнего → блокирующая модалка с warning перед открытием новой сделки
  • Pre-trade checklist — 5 пунктов: SL установлен, размер ≤2%, согласовано с планом, не на эмоциях, R:R ≥2:1. Кнопка «Сохранить» disabled пока не все ✓
  • Tilt Detector — кнопка в ЛК, AI читает дневник, определяет состояние и даёт рекомендацию
  • Психо-журнал — дневник настроения по дням с эмоциями
  • 18 ачивок — мотивация на формирование привычек
💼

Портфель

Учёт реальных позиций с live-ценами.

  • Live-цены с Binance — баланс пересчитывается в реальном времени
  • Allocation pie — кольцевая диаграмма распределения по активам
  • P&L по позиции и общий
  • Архив операций — история покупок и продаж
  • Capital snapshots — точки сохранения для отчётности

Дашборд с кастомными виджетами

16 виджетов на выбор. Собирай главный экран под себя.

  • Funding+OI — перегретые лонги/шорты с auto-fetch
  • BTC Treasuries — публичные holders BTC (Strategy, XXI, etc.)
  • Crypto News — CoinDesk + Cointelegraph (RSS)
  • Economic Calendar — CPI, NFP, PCE, FOMC ближайшие
  • Goal chart, Weekplan, Ideas, Diary — превью из других модулей
  • Paper Trading — баланс + WR%
🎯

Цели и стратегия

Долгосрочные цели и кастомная торговая стратегия.

  • 3 уровня целей — долгосрочные, годовые, месячные с прогресс-барами
  • Стратегия — конфиг + тип рынка (Bull/Bear/Sideways) + правила входа и выхода + таймфрейм + рынки
  • Идеи — трекер торговых идей с категориями и статусами
  • Activity heatmap — 365 дней с подсветкой топ-дня
  • Мой день — план + цели + блоки времени
🌥

Синхронизация и бэкап

3 варианта где хранить данные — выбирай сам.

  • localStorage — по умолчанию, всё на твоём Mac
  • iCloud Drive — авто-синхронизация с AES-256-GCM шифрованием (PBKDF2 200K итераций)
  • Supabase — собственный Postgres для синка между устройствами (включая Windows и web-версию)
  • Export/Import — зашифрованный JSON-бэкап
🔒

Безопасность

Серьёзно подходим к данным.

  • Anthropic API key обфусцирован — XOR + позиционная функция, нет plaintext в бандле
  • CSP whitelist — только разрешённые домены (Binance, Bybit, OKX, CoinGecko, Anthropic, Supabase)
  • Prompt injection защита — данные пользователя в тегах <user_data>
  • Master opt-out AI — выключатель чтобы AI вообще не получал контекст
  • iCloud + Export шифруются AES-256-GCM
