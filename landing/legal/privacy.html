Privacy Policy

Что и куда уходит. Кратко и без воды.

Последнее обновление: 20 мая 2026

1. Главное

Trading Suite — десктоп-приложение для трейдеров: журнал сделок, Paper Trading, AI-ассистент, Funding & OI с трёх бирж, психо-журнал, watchlist с алертами. Десктоп для macOS / Windows + веб-версия. Мы не собираем, не передаём и не имеем доступа к твоим торговым данным.

2. Какие данные ты вводишь

  • Сделки в журнале (тикер, размер, P&L, заметки)
  • Дневник, психо-журнал, эмоции
  • Портфель, watchlist
  • Профиль (имя, аватар, био, торговый стиль)
  • Цели, стратегия, идеи, правила

Все эти данные хранятся только на твоём устройстве и не покидают его без твоего ведома.

3. Внешние сервисы

Приложение делает прямые запросы к публичным API. Мы не проксируем их через свой сервер — твой Mac общается напрямую.

  • Binance, Bybit, OKX — публичные endpoints для цен, funding, OI. Без авторизации, без передачи персональных данных.
  • CoinGecko — публичные данные о BTC treasuries. Без auth.
  • rss2json.com — публичная конвертация RSS (CoinDesk, Cointelegraph). Без auth.
  • Anthropic Claude API — для AI-ассистента. Передаются только те разделы, которые ты явно включил в «Контекст» (см. ниже). Master-выключатель в Личном кабинете отключает контекст полностью.
  • Yahoo Finance (опционально) — для котировок акций и форекса. Без auth.

4. AI-ассистент

Когда ты включаешь «Контекст» в AI-чате, отмеченные тобой разделы отправляются вместе с твоим вопросом в Anthropic. Без твоего явного выбора — ничего не отправляется.

Системный промпт обрамляет твои данные тегами <user_data>, чтобы модель не интерпретировала содержимое как инструкции (защита от prompt-injection).

Если ты хочешь полностью отключить AI-доступ к данным — Master-выключатель в Личном кабинете.

Anthropic обрабатывает запросы согласно их Privacy Policy.

5. iCloud Drive (опционально)

Если ты включаешь синхронизацию через iCloud Drive — данные сохраняются в iCloud Drive/Trading Suite/sync.json. Файл шифруется AES-256-GCM с ключом, который генерируется из твоего пароля (PBKDF2, 200K итераций).

Без пароля файл расшифровать невозможно. Если ты потеряешь пароль — мы не сможем помочь его восстановить.

iCloud — это сервис Apple. Apple имеет свою Privacy Policy.

6. Supabase (опционально)

Если ты подключаешь собственный Supabase проект (по инструкции в Документации) — данные синкаются в твой Postgres. Мы не имеем к нему доступа.

Supabase Auth хранит email и хеш пароля (bcrypt). Trading Suite никогда не видит plaintext пароль.

Связь идёт по TLS. Row Level Security в Postgres гарантирует, что каждый юзер видит только свои строки.

7. Что НЕ собирается

  • Аналитика, метрики, телеметрия — нет
  • Crash reports — нет
  • Уникальные ID устройства — нет
  • Куки, fingerprinting — нет

8. Дети

Приложение не предназначено для лиц младше 18 лет. Трейдинг — деятельность с риском потери средств.
