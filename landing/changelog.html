Changelog

История версий — что добавлено, что поправлено.

v2.5.22 2026-06-27 fix feature
Капитал — исправлена логика общего капитала и добавлена динамика по датам.
  • «Чистый капитал» → «Общий капитал» — пассивы (например, ипотека на квартиру или кредит на авто) больше не вычитаются из общего капитала. Это имущество, которое тоже принадлежит тебе и просто требует расходов на содержание — поэтому теперь общий капитал считается как активы + пассивы, а не активы − пассивы.
  • Структура капитала и сводки — везде, где раньше использовалось «чистый капитал» (карточки, подписи, бары активы/пассивы), теперь подтягиваются обновлённые цифры по новой формуле.
  • Динамика капитала — новый блок на странице Капитал: сохраняй снимок капитала на любую дату и смотри график изменения общего капитала, активов и пассивов во времени, с фильтрами по периоду (1М/3М/6М/1Г/Все) и списком снимков с дельтой к предыдущему.
v2.3.0 2026-05-22 feature
Авторизация через Google и Apple ID.
  • 🔐 OAuth Google и Apple iCloud — кнопки на формах входа и регистрации. Один клик — и ты в аккаунте, без пароля. Реализовано через Supabase OAuth (full-page redirect через провайдера и обратно с токеном).
  • Что нужно для запуска: в Supabase Dashboard → Authentication → Providers нужно включить Google и Apple, прописать ключи из Google Cloud Console и Apple Developer Portal.
  • В Electron-десктопе кнопки скрыты (full-page redirect не работает), там доступен email/password.
v2.2.5 2026-05-22 fix
Гости теперь могут открывать любые страницы приложения.
  • 🔓 Funding + OI, Стратегия и другие Pro-страницы открываются для гостей — раньше клик показывал toast «Pro feature» и редиректил на профиль. Теперь гости видят содержимое и могут оценить функционал до регистрации. Лимиты на действия (например, 5 сообщений AI) остаются.
v2.2.4 2026-05-22 feature
Туториал теперь запускается и для гостей.
  • 🎓 Onboarding-тур для гостей — раньше 7-шаговый тур стартовал только после регистрации, теперь запускается при первом визите в любом режиме. Гость сразу знакомится с фичами и может решить регистрироваться или нет, видя ценность.
  • Guard оставлен только на случай открытого auth-gate модала (сброс пароля, подтверждение email) — там туториал был бы закрыт другим окном.
v2.5.21 2026-06-27 feature release
Капитал, металлы и обновленные desktop/web builds.
  • Калькулятор металлов — бывший калькулятор золота переименован и расширен: добавлено серебро, переключатель Gold/Silver, live-цены XAU/XAG, USD/EUR, стандартные слитки и резервные источники данных.
  • Капитал — новый раздел для учета всех классов имущества: недвижимость, акции, облигации, золото, серебро, авто, бизнес, наличные, кредиты, ипотека и другие активы/пассивы.
  • Структура капитала — добавлены KPI по активам, пассивам и чистому капиталу, donut-диаграмма по классам активов и сравнение Assets vs Liabilities.
  • Портфель Криптовалюты — старый раздел «Портфель» переименован, чтобы отделить криптопозиции от общей картины капитала.
  • Release files — собраны и опубликованы macOS DMG и Windows ZIP 2.5.21, обновлена web-версия и download-ссылки.
v2.2.3 2026-05-22 feature fix
Гостевой режим + AI триал + UX-полировка.
  • 🤖 AI триал для гостей — 5 бесплатных сообщений до регистрации. На 6-й попытке всплывает модалка с предложением зарегистрироваться (раньше AI был полностью заблокирован для гостей).
  • 🎯 Стратегии в Paper Trading — отдельная вкладка для гипотез: создание, редактирование, удаление прямо с карточки. Аналитика по каждой стратегии (винрейт, P&L, profit factor). При открытии позиции — dropdown выбора стратегии.
  • ← На сайт — кнопка возврата на лендинг в топбаре (только в веб-версии).
  • Подсветка активной вкладки в Paper Trading — фиолетово-розовый градиент вместо еле различимой подсветки.
  • Полные переводы Paper Trading на EN — стратегии, аналитика, модалки, тосты, плейсхолдеры.
  • Auth UI fix — добавлен pre-paint state и reconciliation на focus/visibilitychange, чтобы плашки регистрации не залипали в «limbo» состоянии при гонке с восстановлением сессии.
  • Точные тексты в гостевом режиме — «Data is only in this browser» вместо «Changes won't be saved». Логин нужен для синка между устройствами, локально данные сохраняются.
v2.5.22 — 2.5.22 2026-05-20 feature fix
Pre-launch polish — большой блок улучшений.
  • 🎓 Onboarding-туториал — 7-шаговый тур при первом запуске с переключателем RU/EN, deep-link на каждую фичу. После клика на кнопку «Открыть …» вместо закрытия показывается плавающая пилюля «Продолжить туториал →»; resume сохраняется между запусками. Кнопка перезапуска в Профиле → Бэкап.
  • Темная тема — контраст — --text3 поднят с #55556a (3.5:1, ниже WCAG AA) до #8484a8 (~6:1). Таймштампы, placeholders, «No entries» — комфортно читаются.
  • Дневник: вкладка «Все записи» + исправление даты, которая выглядела «скукоженной». «+ Новая запись» и клик по записи автоматически переключают на форму.
  • EN-перевод — fixes для «В архиве no trades», «Take profit — необяз.», «Введи параметры», виджеты «Эконом. календарь / Крипто новости» теперь перерисовываются при смене языка.
  • Страница «Стратегия» — fix sticky .st-editing класса (после Сохранить × и + кнопки оставались видимыми). Защита от потери русских текстов при сохранении в EN-режиме через __origText и auto-heal при загрузке.
  • Persistence audit — починены: лічильник «Макс. сделок» в Торговом плане, теги инструментов/запрещённых, цель/заметки (autosave). Goal-chart auto-init теперь сразу пишет в gc_v1, delete-неделя сохраняет полный стейт. Fix AI ctx key mismatch (psy_v1 → psycho_v1).
  • Дата в Ежедневнике — теперь по центру и во всю ширину ячейки.
