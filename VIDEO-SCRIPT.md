# 🎬 Trading Suite — сценарий видеообзора

**Длительность:** 3 мин 30 сек (адаптируется до 60 сек / 5 мин)
**Язык:** Русский (EN-версия — копия с заменой на конец файла)
**Темп речи:** ~150 слов/мин, разговорный
**Формат записи:** `Cmd+Shift+5` → запись окна или области (16:9, 1920×1080 рекомендую)

---

## 🛠 Pre-flight Checklist (за 10 минут до записи)

### Окружение
- [ ] Закрыть все лишние приложения (особенно мессенджеры — уведомления попадут в кадр)
- [ ] **Тёмная тема** — выглядит дороже на видео (в Trading Suite по умолчанию темно)
- [ ] **Скрыть Dock** (Cmd+Opt+D) и **menu bar** на время записи (или System Settings → Control Center → автоскрытие)
- [ ] Размер окна **1280×860** (дефолтный) — комфортно для 16:9
- [ ] Язык приложения **RU** (Личный кабинет → язык)

### Данные для красивых скриншотов
Чтобы UI не выглядел пустым:

- [ ] **Портфель**: добавить 3-4 позиции (BTC, ETH, SOL, AVAX) — уже есть в seed-данных
- [ ] **Журнал**: 8-10 сделок за последний месяц (микс прибыльных/убыточных) — для аналитики и графика equity
- [ ] **Дневник**: 3-5 записей за неделю — для тильт-детектора
- [ ] **Психо-журнал**: 5-7 дней с эмоциями
- [ ] **Paper Trading**: 1-2 открытые позиции (BTC long, ETH short) — чтобы Equity не был 0
- [ ] **Watchlist**: 5-7 монет, у 2-3 выставить алерт (для визуала)
- [ ] **Цели**: заполнить 1 долгосрочную, 2 годовых, 3 месячных
- [ ] **Стратегия**: написать пару правил входа/выхода

### Технические
- [ ] **Курсор крупный** — System Settings → Accessibility → Pointer → Size 2-3x
- [ ] **Plain mouse** (без свайпов между рабочими столами)
- [ ] **Не клацай быстро** — пауза 1-2 секунды на каждом экране для читаемости
- [ ] **Не моргай курсором** — спокойные движения

## 🎙 Озвучка через `say` (опционально, если не хочешь сам)

macOS встроенный TTS. Качество — Siri-голос, нормальное.

```bash
# Сгенерить аудио из voice-over.txt
say -v Milena -r 165 -o voiceover.aiff -f voice-over.txt

# Список русских голосов
say -v '?' | grep -i ru
```

**Голоса:** `Milena` (RU, чётче) или `Yuri` (RU, мужской). Скорость `-r 165` — комфортный темп.

Для лучшего качества — ElevenLabs / OpenAI TTS (платные API, но звучат живее).

## 🎬 Монтаж (после съёмки)

### Минимальный (без редактора)

`Cmd+Shift+5` → записал → готовый `.mov` в `~/Desktop`. Залить как есть, если получилось хорошо.

### Если нужна обрезка/склейка

**Без установок:** QuickTime Player → Edit → Trim или Split Clip.

**С ffmpeg** (после `brew install ffmpeg`):

```bash
# Обрезать от 0:05 до 3:35
ffmpeg -i input.mov -ss 00:00:05 -to 00:03:35 -c copy output.mov

# Ускорить кусок в 2 раза (для скучных моментов)
ffmpeg -i input.mov -filter:v "setpts=0.5*PTS" -an output.mov

# Наложить voice-over.aiff на видео без оригинального звука
ffmpeg -i video.mov -i voiceover.aiff -map 0:v -map 1:a -c:v copy -c:a aac output.mp4

# Конвертировать .mov → .mp4 для YouTube/Twitter
ffmpeg -i input.mov -c:v libx264 -crf 20 -c:a aac output.mp4
```

### Если нужен профи-монтаж

- **DaVinci Resolve** (бесплатно) — лучший free редактор, есть для Mac
- **iMovie** — встроен в macOS, простой
- **Final Cut Pro** — если уже куплен
